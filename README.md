# プライバシーポリシー — Headshot

最終更新日: 2026-06-07

## データ収集について

Headshot はユーザーの個人データを一切収集せず、外部サーバーへのデータ送信も行いません。

## ローカルへの保存

- 設定（キー設定・インターバル・保存先フォルダ等）は `chrome.storage.local` を使用してお使いのデバイス上にのみ保存されます。
- キャプチャした画像は Chrome Downloads API を通じてお使いのデバイスのダウンロードフォルダに直接保存されます。
- いかなるデータもデバイス外部に送信されません。

## 第三者サービス

Headshot はアナリティクス・広告・トラッキングサービスとの連携を一切行いません。

# Privacy Policy — Headshot

Last updated: 2026-06-07

## Data Collection

Headshot collects no personal data and sends nothing to external servers.

## Local Storage

- Settings (key bindings, interval, save folder, etc.) are stored in `chrome.storage.local` on your device only.
- Captured images are saved directly to your device's download folder via the Chrome Downloads API.
- No data is transmitted outside your device.

## Third-Party Services

Headshot does not integrate with any analytics, advertising, or tracking services.
